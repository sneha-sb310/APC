# Arithmetic Precision Calculator (APC)

## Overview
This project is an Arithmetic Precision Calculator developed during training at Emertxe Information Technologies.

It performs arithmetic operations with high precision, handling large numbers and accurate calculations beyond standard data type limits.

---

## Features
- Supports basic arithmetic operations (Addition, Subtraction, Multiplication, Division)  
- Handles large numbers with high precision  
- Implements custom logic instead of relying on built-in data types  
- Structured and modular implementation  

---

## Technologies Used
- C Programming  
- GCC Compiler  

---

## How to Run
```bash
gcc main.c -o apc
./apc
```

---

## Learning Outcomes
- Understanding of precision-based arithmetic operations  
- Implementation of custom logic for large number handling  
- Improved problem-solving and coding skills
